## ElementRect

An object defining the Element Rect.

**PROPERTIES**

- <code>x: number</code>
- <code>y: number</code>
- <code>width: number</code>
- <code>height: number</code>

## LocatorStrategy

Strategy for searching element on the page

**POSSIBLE VALUES**

- <code>'css selector'</code>
- <code>'link text'</code>
- <code>'partial link text'</code>
- <code>'tag name'</code>
- <code>'xpath'</code>
